# 🌟 3-View Login Page

A stunning, premium, and fully responsive authentication and dashboard UI template utilizing modern glassmorphism design principles. Built purely with HTML and CSS for top-tier aesthetics and lightning-fast load times.

> **Project Creation Date:** July 31, 2024

---

## ✨ Features

- **💎 Glassmorphism Design Theme**: Semi-transparent backgrounds, blur effects, and rich gradients that float beautifully.
- **📱 Fully Responsive**: Built to adapt beautifully to screens of all sizes, from mobile devices to large desktops.
- **🖋 Modern Typography & Icons**: Uses Google's `Inter` font for extreme legibility and `BoxIcons` for sleek visual cues.
- **✨ Micro-interactions**: Smooth hover effects, floating input labels, and subtle transitions make the interface feel alive.
- **🚀 Plug and Play Setup**: No complicated frameworks required.

---

## 🎨 Pages Included

1. **`index.html`** - A welcoming Hero/Landing page acting as the gateway to the app.
2. **`sign_in.html`** - A secure-looking login portal with floating labels and an elegant gradient action button.
3. **`sign-up.html`** - An intuitive account creation form.
4. **`forgot_password.html`** - A clean fallback procedure for password resets.
5. **`dashboard.html`** - A conceptual logged-in state showcasing statistics and account options.

---

## 🚀 Getting Started

Simply run the code directly in your browser:

1. Clone or download this repository.
2. Open `index.html` in your favorite web browser.
3. Click through the links to navigate the entire beautifully linked flow!

---

## 🎨 Customization

You can easily change the global color scheme of the entire project to match your custom brand palette. Open `style.css` and simply modify the `:root` variables:

```css
:root {
  --primary-color: #f410f4;
  --secondary-color: #940e94;
  --accent-color: #791bf5;
}
```
