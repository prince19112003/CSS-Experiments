# 🧪 UI/UX CSS Laboratory

A specialized collection of my creative frontend UI components, advanced CSS animations, and interaction design experiments. 

This repository serves as a showcase of modern vanilla CSS capabilities, bypassing heavy JavaScript calculations to produce visually striking and highly performant user interfaces.

## 🌟 The Experiments Include:

1. **Glow Liquid Form:** Multi-layered spectral gradients using `@keyframes` mapped behind a button for an animated, moving neon background.
2. **Glassmorphism Interaction:** Real-time blurred backdrops mimicking frosted glass using CSS `backdrop-filter: blur()`, paired with delayed cubic-bezier revealing transitions.
3. **Neumorphic Deep Toggle:** Soft UI relying heavily on multi-layered internal and external box shadows to create authentic 3D depths that react securely to the CSS `:checked` pseudo-class.
4. **Fluid SVG Filter Loader:** A "gooey" animation built by combining CSS translations with an injected SVG `<feColorMatrix>` filter that organically blends moving div nodes.
5. **Background Mapped Typography:** Text rendered completely transparent while using `-webkit-background-clip: text` to mask an animated, continuously oscillating hyper-gradient backing.
6. **Perspective Tilt Shift Button:** A hardware-accelerated 3D object employing `perspective` bounds to calculate dramatic X/Y rotations with underlying volumetric shadow projections.

## 🚀 How to Use
No frameworks, no compilers, no node modules.
Simply pull the repository to your machine and open `index.html` in any modern web browser to explore the component lab dynamically.
