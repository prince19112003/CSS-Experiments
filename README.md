Welcome to the **UI/UX CSS Laboratory**. This repository is a specialized collection of creative frontend UI components, advanced CSS animations, and interaction design experiments. It showcases the power of pure Vanilla CSS to produce visually striking and highly performant interfaces.

---

## 📂 Featured Experiments

### 1. 🎨 CSS Visual Elements
*The main laboratory of stylistic components.*
- **Highlights**: 6+ advanced CSS visual experiments including custom inputs, buttons, and state-driven layouts.
- **Goal**: Demonstrating intricate styling without external dependencies.
- [📁 Explore Code](./Css-Elements)

### 2. 🖱️ Aesthetix Cursor Worm
*Dynamic, responsive cursor interaction.*
- **Highlights**: An organic, "worm-like" cursor trailing effect that follows the user's focus with smooth physics.
- **Goal**: Enhancing user engagement through fluid, interactive feedback.
- [📁 Explore Code](./Cursor-worm)

### 3. 💓 Life-Pulse Animation
*Infinite heartbeat rhythm.*
- **Highlights**: A biologically accurate, continuous heartbeat pulsing CSS animation using advanced `@keyframes`.
- **Goal**: Exploring complex timing functions and scaling transforms.
- [📁 Explore Code](./Heart-Beat-animation)

### 4. ✨ Magical Wish Box
*The ultimate interactive gift experience.*
- **Highlights**: A multi-stage opening sequence for a digital gift box, featuring smooth transitions and particle-like effects.
- **Goal**: Designing an emotional and rewarding user interaction flow.
- [📁 Explore Code](./Wish-Box)

### 5. 🔑 3-View Premium Auth
*High-end authentication interface designs.*
- **Highlights**: Detailed sign-in, sign-up, and forgot-password views with glassmorphism and modern layout patterns.
- **Goal**: Providing a professional-grade baseline for secure and inviting entry points.
- [📁 Explore Code](./3-view-login-page)

## 🛠️ Laboratory Tech Stack

| Feature | implementation |
| :--- | :--- |
| **Logic** | Pure Vanilla JavaScript (zero frameworks) |
| **Animations** | CSS Keyframes, Hardware-Accelerated Transforms |
| **Styling** | Custom Properties (Variables), Flexbox/Grid Mastery |
| **Visuals** | Backdrop Filters (Glassmorphism), Complex Gradients |

## 🚀 Getting Started

No configuration required. This is a pure vanilla web stack.

1. **Clone the repo:**
   ```bash
   git clone https://github.com/prince19112003/CSS-Experiments.git
   ```
2. **Launch:**
   Open any sub-directory and double-click the `index.html` file to view the experiment in your browser.
