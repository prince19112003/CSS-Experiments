## 🌟 About This Project

This project explores the power of **pure CSS animations** to create organic, life-like movements. By carefully tuning bezier curves and applying intensive keyframes, this heart doesn't just simply scale up and down—it performs a genuine human double-beat.

### Key Highlights:
- **Intricate Keyframes**: Custom CSS `@keyframes` orchestrating the true human pulse rhythm.
- **Neon Aesthetics**: Utilizes CSS `filter: drop-shadow()` for a seamless, radiating neon glow against a dark radial canvas.
- **Energy Rings**: Expanding and fading pulse rings that synchronize perfectly with the heart's expansion.

<br/>

## 🚀 Getting Started

To view the animation locally, you don't need a development server or any dependencies. 

1. Clone or download the repository.
2. Open `index.html` in any web browser.
3. Enjoy the visuals!
